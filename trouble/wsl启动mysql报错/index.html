wsl启动mysql报错

后端 linux wsl 2020/05/19

报错信息
mkdir: cannot create directory ‘//.cache’: Permission denied

解决方案

刪掉/etc/profile.d/wsl-integration.sh就行了

命令为:
sudo rm -rf /etc/profile.d/wsl-integration.sh
